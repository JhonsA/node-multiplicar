# Instrucciones
```
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Especifica el número base de la multiplicación
                                                             [number] [required]
  -l, --listar   Muestra la tabla en consola          [boolean] [default: false]
  -h, --hasta    Especifica hasta que valor quiere multiplicar
                                                          [number] [default: 10]
```

## Para reconstruir modulos de node
```
npm install
```
